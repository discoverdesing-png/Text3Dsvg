# Text3Dsvg - Editor 3D Online
by Pedro Simental Jacinto

Crea texto 3D y expórtalo como PNG o video.

👉 [Úsalo aquí](https://discoverdesing-png.github.io/Text3Dsvg/)

### Features
- Texto 3D personalizable
- Exporta PNG con transparencia 
- Graba videos de 5 segundos
- Controles de luz, material, profundidad
